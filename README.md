# SyncPlay - MVP

Projeto acadêmico: versão simplificada do SyncPlay para entrega da Etapa 2.

## O que tem aqui
- Expo React Native skeleton
- Telas incluídas:
  - Login (email + senha)
  - Signup (email + senha + confirmar)
  - Feed (lista de posts)
  - Search (placeholder para categorias)
  - Player / detalhes (abre post)
  - Profile (logout)

## Como executar localmente
1. Instale o Expo CLI: `npm install -g expo-cli`
2. No diretório do projeto: `npm install`
3. Execute: `npm start` (ou `expo start`)
4. Abra no app Expo Go no celular escaneando o QR code.

> Observação: este projeto usa AsyncStorage para salvar usuários localmente (mock). Não há backend.

## Entrega acadêmica
- Subir este repositório no GitHub como `syncplay`
- Para o MVP funcional para testes de usuário, use Glide (low-code) com as mesmas telas e dados.
